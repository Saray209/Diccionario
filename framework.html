React

React es una biblioteca de JavaScript para construir interfaces de usuario. Fue desarrollada por Facebook y es ampliamente utilizada para construir aplicaciones web modernas.

Características Principales:

  • Componentes Reutilizables
  • Virtual DOM
  • Unidirectional Data Flow
  • JSX (JavaScript XML)
